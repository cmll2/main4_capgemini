Il faut utiliser le programme de cette façon :

dans la ligne de commande, si vous voulez utiliser juste un csv :

python3 rt_script_classes.py *chemin vers votre csv*

dans la ligne de commande, si vous voulez utiliser un modèle déjà enregistré via joblib :

python3 rt_script_classes.py *chemin vers votre modèle* *chemin vers vos args*

--> *args* sont les arguments nécessaires extraits en même temps que votre modèle lors de la première utilisation avec le csv

Voila !
